# 📝 To-Do List App

A simple and elegant To-Do List web application built with **HTML**, **CSS**, and **JavaScript**.

## ✨ Features
- ✅ Add tasks
- ❌ Delete tasks
- ✅ Mark completed
- 💾 Save tasks in browser (localStorage)

## 🚀 How to Use
1. Type your task in the box
2. Click **Add**
3. Click task to mark as done ✅
4. Click ❌ to delete
